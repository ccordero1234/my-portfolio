PickUp

Pickup is a mobile app designed to connect people who want to play pickup sports. It allows users to create profiles, add friends, track game stats, and view leaderboards. The app also integrates Google Maps so players can find games nearby and form teams. Built with React Native and Firebase, Pickup focuses on combining social features with sports to make it easier for people to meet, play, and compete.
